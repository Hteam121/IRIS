---
name: focus-mode
description: Clear distractions and get the user set up to work.
mode: inline
tools:
  - open_app
  - press_key
---
1. Ask which project or app they want to focus on if they didn't say.
2. Open their editor or the app they named.
3. Hide other distracting apps (activate each and press command+H), starting with obvious
   ones like Messages, Slack, or Mail if they're visible.
4. Confirm in one short sentence that focus mode is ready.
