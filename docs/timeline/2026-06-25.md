## 19:10 — [Cost] CostGovernor — budget-driven cost control (target ≤ $20/mo)
- Goal: cut a ~$100/mo realtime-heavy user toward a user-set cap by (a) using free `claude -p` for
  every text-brain call and (b) heavily optimizing OpenAI. The cap is a Setting; IRIS adapts to it
  dynamically rather than via a fixed mode.
- New `IRIS/Core/CostGovernor.swift`: `@MainActor` meter — exact per-turn cost from the Realtime
  `usage` object, `gpt-4o-mini-tts` char cost, `~/.iris/usage.json` persistence with monthly/daily
  rollover, and a `.premium`/`.saver`/`.free` `tier()` driven by spent-fraction (≥75% → saver) plus
  daily pacing (`remaining ÷ days-left`). See docs/algorithms.md → Cost Governor for rates/thresholds.
- Settings: added `monthlyBudgetUSD` (default $20, 0 = unlimited; env `IRIS_MONTHLY_BUDGET`) wired
  through load/save/init/`withUpdatedKeys`; **default realtime model `gpt-realtime` → `gpt-realtime-mini`**
  (~3.2× cheaper audio).
- Wiring: `RealtimeSession` meters on `response.done`; `Speaker` gates neural TTS off in `.free`
  (→ on-device) and meters chars; `AppDelegate` creates the governor, injects it, and tier-gates at
  `wakeUp` — premium opens realtime, saver/free leave the wake detector live so the captured command
  is answered by the free classic `claude -p` pipeline (wired `onWakeWordDetected`/`onBargeIn` on the
  realtime-mode detector for this).
- Route everything else to `claude -p`: `IntentRouter` now prefers a `claude -p` JSON classifier over
  gpt-4o for ambiguous commands; `RealtimeTools.look_at_screen` uses `claude -p --allowedTools Read`
  (gpt-4o only as fallback when the binary is missing).
- UI: `SettingsView` gained a "Monthly OpenAI budget (USD)" field; replaced the earlier `budget: ""`
  bridge in `AppDelegate.showSettings` with the real value.
- Files: `Core/CostGovernor.swift` (new), `Config/Settings.swift`, `Realtime/RealtimeSession.swift`,
  `Realtime/RealtimeTools.swift`, `Voice/Speaker.swift`, `AI/IntentRouter.swift`, `AppDelegate.swift`,
  `UI/SettingsView.swift`, `docs/algorithms.md`.
